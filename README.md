# ServerPulse

Dashboard web standalone para monitorear homelabs Docker: contenedores, métricas del sistema y estado de servicios — sin depender de un backend específico.

Se conecta a través de adaptadores configurables: HomeCore API, Prometheus, o directo contra el socket de Docker.

> 🚧 En desarrollo — Fase 1: setup del proyecto y capa de adaptadores.

## Stack
- React 18 + TypeScript + Vite
- TailwindCSS
- Recharts
